JSON, Objects and Stringify examples

JSON or JavaScript Object Notation is the most used way to store and transport data. This use of objects and its syntax makes it pretty self describing. There is also the method Stringify that converts objects to JSON and JSON to objects! meaning you don't have to write much. Each attribute/element must be within brackets . Each object will be encapsulated in {} while arrays are encapsulated in []

Note that you may also use a JSON editor online that will activly let you know when you make a syntax error.
