Var - JavaScript

JavaScript uses the "var" keyword to declare variables. These do not need to be initialized or a data type stated. This is commonly known as a dynamic type. This aslo means that you can change an int into a random string without having to redifine the variable.

We must take note of this when doing semi arithmatic operations such as +.
For example 2016 + 1 + "Wollongong" Will equal 2017Wollongong.
To bypass this you can use the toString(); command.

JavaScript also has the boolean data type. This can be dynamically named as a mathamatical statement.
var positive = (x > 0); //true

String dataType

The string data type does not need to explicitly stated when the varaible is defined. This is assigned automatically when " " are used.

String index starts at 0, multiple methods can be used surround strings.

  • .length();
  • .toUpperCase();
  • .toLowerCase();
  • .indexOf("content");
  • .includes("content");
  • .slice(a,b);

Date DataType

This variable is an object, therefore it must be instantiated. This can be done by:
